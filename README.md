# PictureList

このアプリは、画像ファイルの EXIF / メタデータを簡単に表示するための Windows フリーウェアです。

## 特徴
- Excelに張り付ける、Exifを含んだファイル情報を作成
- 任意のローカルフォルダ以下のファイルリストを作成
- Exifを表示しなければ高速動作
- 写真のExif情報でMakerNoteなどを除き実際のデータはほぼすべて選択可能
- Exif情報はExifの日本語の表示を採用
- ExifTool でファイル情報を表示可能（同梱）
- ExifToolはクラスを利用しているので、ExifToolの追加データを利用がプログラムを改造すれば比較的簡単に追加可能
- 外部通信なし（完全オフライン動作）
- ソースコード公開
- 並列処理化による高速化は、自信がないので現時点では未実装

## ダウンロード
最新版の MSI インストーラーは以下からダウンロードできます：
https://github.com/KenichiUehara/PictureList/releases

## 使い方
1. MSI を実行してインストール
2. アプリを起動
3. 画面に従ってフォルダーを選択し実行
4. 画面をみればほぼ使い方がわかると思います
5. 探索を途中でキャンセルできます
6. アクセス権がない場合などのエラーログは員スールフォルダの PictureList.Log に出力（サイズは自動切詰）
