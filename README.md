# PictureList

このアプリは、画像ファイルの EXIF / メタデータを簡単に表示するための Windows フリーウェアです。

## 特徴
- EXIF 情報の高速読み取り
- ExifTool を内部で使用（同梱）
- 外部通信なし（完全オフライン動作）
- ソースコード公開

## ダウンロード
最新版の MSI インストーラーは以下からダウンロードできます：

👉 https://github.com/ユーザー名/PictureList/releases

## 使い方 }
1. MSI を実行してインストール
2. アプリを起動
3. 画像をドラッグ＆ドロップ
